La déclaration de revenus est une obligation légale pour tous les citoyens français et étrangers résidents fiscaux en France. Elle permet à l'administration fiscale de calculer l'impôt sur le revenu (IR) que vous devez payer.

Déclaration pour les non-résidents :

Si vous êtes un non-résident fiscal français, vous devez remplir une déclaration de revenus spécifique. Vous pouvez la trouver sur le site impots.gouv.fr.


Déclaration pour les micro-entrepreneurs :

Les micro-entrepreneurs (anciennement auto-entrepreneurs) ont un régime fiscal simplifié. Ils doivent déclarer leurs revenus sur une déclaration spécifique accessible sur le site autoentrepreneur.urssaf.fr.


Paiement des impôts :

Vous pouvez payer vos impôts en ligne, par prélèvement automatique ou par chèque.


Déclaration des revenus exceptionnels :

Si vous avez perçu des revenus exceptionnels (ex : plus-value immobilière), vous devez les déclarer sur une déclaration spécifique.

Déclaration des frais professionnels :

Si vous avez des frais professionnels déductibles, vous devez les déclarer sur une déclaration spécifique.

notes: >- alt: >- credit: >- - kind: heading title: >- Comment peut-elle se faire ? position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

-Déclaration en ligne : La méthode la plus simple et rapide.

-Déclaration papier : Disponible pour les personnes qui ne peuvent pas ou ne souhaitent pas déclarer en ligne.

-Déclaration préremplie : L'administration fiscale remplit une partie de la déclaration pour vous. Vous devez la vérifier et la compléter si nécessaire.

notes: >- alt: >- credit: >- - kind: heading title: >- Etapes clés : position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Remplir la déclaration de revenus : Vous pouvez la faire en ligne sur impots.gouv.fr ou sur papier.

Déclarer tous vos revenus : Salaires, pensions, revenus fonciers, etc.

Joindre les justificatifs : Avis d'impôt sur le revenu de l'année N-2, justificatifs de charges déductibles, etc.

Payer l'impôt : Si vous avez un solde à payer, vous pouvez le régler en ligne, par prélèvement automatique ou par chèque.

    • Vos charges déductibles : Frais professionnels, pensions alimentaires, etc.
    • Votre situation familiale : Nombre d'enfants à charge, mariage, etc.
